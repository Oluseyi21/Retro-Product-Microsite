# Retro Product Microsite – Polaroid Camera
# Project Overview

This project is a responsive product microsite inspired by a retro-style Polaroid camera brand. The website was designed to showcase a product through clean layout structure, engaging visuals, and interactive features while ensuring compatibility across multiple devices and browsers.

The project demonstrates front-end development fundamentals, responsive design principles, and structured testing practices.

# Technologies Used

HTML5

CSS3

JavaScript

Git & GitHub

# Key Features

Fully responsive layout for desktop, tablet, and mobile devices

Structured design using Flexbox and CSS Grid

Interactive JavaScript elements to enhance user engagement

Clean and organized UI structure

Cross-browser compatibility testing

Version control using GitHub

# Project Highlights

Successfully built a responsive microsite, improving accessibility across screen sizes.

Effectively implemented interactive functionality using JavaScript, enhancing user experience.

Carefully tested layout and features across multiple browsers to ensure consistent performance.

Organized project structure and version history using GitHub.
